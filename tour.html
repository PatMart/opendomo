Can you imagine a world where all devices can live together in harmony?

A world of efficiency, where modern technology makes sense?

Well, then this is possible nowadays, and it's in fact our dream, and OpenDomo's vision.

OpenDomoOS is the first free operative system specially designed to provide an accessible and secure control system for home automation. And a lot more.

The license under which the project is being developed the General Public License (GPL). This license guarantees freedom in the use of software, a core value of the current technological society. In general terms, this means that OpenDomo can be used, copied, modified and distributed freely.

But how does it work?

  • OpenDomoOS is the cornerstone of the electronic appliances
  • It takes advantage of modern communication protocols, such TCP/IP
  • It connects all supported devices and controls them in an homogeneous way
  • This is the only possible way to become truly efficient
  • It's of course based on standards, and open protocols

OpenDomoOS provides a responsive interface that enables virtually any modern device (tablet, phone or computer) to control the appliances.

Yes, you are free to choose!

But here is the twist...

All these characteristics are permanently growing because OpenDomoOS support plugins, each one of them provides new incredible features*

* and anyone is free to create them

For instance, odAutomation provides compatibility with ODControl and Arduino devices, and allows you to configure scenes and sequences:

  • "Turn all the lights off"
  • "Activate theater mode"
  • "Close all blinds and awnings"

The plugin IVI (or Interactive Visual Interface) allow control of all devices from a unified graphical view.

 * this plugin is in Beta mode

With odMusic all the MP3 files can be organized in playlists and streamed as background music anytime

 * this plugin is in Beta mode

Viewing all the cameras of the house at once, it's very easy with odVision. And very soon:

  • Recording
  • Surveillance
  • Artificial Vision
  • ...
 * this plugin is in Beta mode
