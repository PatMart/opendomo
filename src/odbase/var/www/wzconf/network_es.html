Configuración de red

Inicio de sesión

Antes de empezar, es necesario iniciar sesión en el agente de OpenDomo. Para hacerlo, haz click aquí y escribe "admin" como nombre de usuario y "opendomo" como contraseña en la ventana de inicio de sesión (o la correspondiente, si la has cambiado).

Configuración de red

Si has iniciado el equipo por primera vez, tal vez la configuración de red obtenida no es la deseada. Por defecto el sistema asigna la IP 192.168.2.2, pero es posible emplear una configuración personalizada. Si deseas modificar la configuración haz click aquí

Guardar la configuración

Una vez hechos los cambios esenciales, ya puedes guardar la configuración haciendo click aquí.
