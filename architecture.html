OpenDomoOS

The free OS for home automation

View project onGitHub

Overview

Notification
The system can be configured to send notifications when certain events take place
 Alarms
Visual and auditive alarms are useful to warn the user when something important happened
 Music
OpenDomoOS can use your MP3 files as background music
 Blinds
Adjust blinds, shades and awnings automatically when it's too warm to keep your home fresh
 Lights
Save energy using the lights only when necessary
Switches
Standard switches and push-buttons can be used with OpenDomoOS
 Sensors
Temperature, humidity and light sensors will help to make a more comfortable space
 Camera
IP cameras are easy to install and they will provide the safety that your home needs
 Microphone
Voice commands can be used to control the whole system
Cloud Network
OpenDomoOS uses the standard IP network and it doesn't need any dedicated cabling; this makes it an even more affordable solution
 Meter
The ODEnergy meter is an excellent tool to measure the energy consumption in real time
 Controller
Each ODControl controller can handle up to 8 switches, 8 electric lines, 2 dimmers and 2 sensors, and you can stack as many as you need!
 Controller
The ODControl controller can be installed anywhere in the house; it just need access to the network to connect in cascade with the other ODControl!
 Network
Using the IP network has multiple advantages, but the most important is that all the modern houses already have it deployed
Desktop
Using standard technologies as HTML5 has the advantage that all modern devices are compatible with it
 Laptop
From your home, at work or during your holidays, you can make sure that everything is going right
 Cellphone
If you prefer, there are also Apps for Android and iPhone with extended features
 Tablet
Try it on a tablet if you can; the interface is optimized for touch devices and it works great!
